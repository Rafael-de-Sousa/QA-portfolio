# Sistema de Agendamento - Clínica Médica

Sistema web desenvolvido para simular o agendamento de consultas médicas, com foco em testes manuais, documentação e boas práticas de QA.

---

## Objetivo do Projeto

Este projeto foi criado para compor meu **portfólio de QA**, demonstrando:

- Planejamento de testes
- Criação de casos de teste
- Reporte de bugs
- Execução de testes manuais
- Organização de documentação
- Simulação de ciclo de testes real

---

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- localStorage (armazenamento simulado)

---

## ⚙️ Funcionalidades do Sistema

Cadastro de consulta médica  
Listagem de agendamentos  
Remoção de consultas  
Persistência de dados no navegador  
Validação básica de campos  
Verificação de conflito de horário

## Bugs Identificados

Durante a execução dos testes foram encontrados problemas como:

- Cadastro com campos vazios
- Falta de validação de data passada
- Possíveis inconsistências no armazenamento

Todos os bugs foram documentados na pasta:

```
bug-reports/
```
