# Projeto QA — Loja Virtual
https://rafael-de-sousa.github.io/Loja-Virtual/

## Sobre o Projeto

Este projeto foi criado com o objetivo de demonstrar minhas habilidades em
Qualidade de Software (QA), aplicando testes manuais, documentação e boas práticas
em um sistema de e-commerce.

O sistema simula uma loja virtual com funcionalidades de cadastro, login, carrinho
e checkout.

---

## Objetivo

Demonstrar minha capacidade de:

- Criar casos de teste funcionais
- Elaborar planos de teste
- Registrar bugs de forma profissional
- Organizar documentação de QA
- Seguir padrões utilizados no mercado

## Gerenciamento de Defeitos

Os defeitos encontrados durante os testes estão documentados na pasta:

bug-reports/


Cada bug contém descrição, passos para reprodução, severidade e evidências.
