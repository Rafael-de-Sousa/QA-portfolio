# Automação Cypress - Loja Virtual

Projeto de automação de testes E2E para uma loja virtual.

## Tecnologias
- Cypress
- JavaScript
- Node.js
- Bootstrap

## Como executar

```bash
npm install
npx cypress open
```

## Cenários Automatizados

Adicionar produto no carrinho

Remover item

Limpar carrinho

Finalizar compra

Validação do formulário

## Estrutura
cypress/e2e/loja.cy.js
