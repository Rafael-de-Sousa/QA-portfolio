Bug Report - Erro no Login

---

Informações Gerais

- **ID:** BUG-001  
- **Título:** Usuário não consegue realizar login com credenciais válidas  
- **Módulo:** Login  
- **Ambiente:** Web - Chrome / Windows 10  
- **Versão:** v1.0  
- **Data:** 30/01/2026  
- **Reportado por:** Rafael  

---

Descrição

Ao tentar realizar login com usuário e senha válidos, o sistema retorna mensagem de erro e não permite acesso.

---

Passos para Reproduzir

1. Acessar a página de login  
2. Inserir usuário válido  
3. Inserir senha válida  
4. Clicar em **Entrar**

---

Resultado Esperado

O sistema deve autenticar o usuário e redirecionar para a área logada.

---

Resultado Atual

O sistema exibe a mensagem:

> "Usuário ou senha inválidos"

Mesmo com dados corretos.

---

Evidências

Anexar print da tela (se houver).

Severidade e Prioridade

- **Severidade:** Alta  
- **Prioridade:** Alta

---

Observações

O erro ocorre apenas no navegador Chrome.
